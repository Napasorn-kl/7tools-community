เชื่อมต่อ Google Sheets

วาง URL ที่ได้จากการ Deploy Apps Script (ดูขั้นตอนด้านล่าง)

📋 วิธีตั้งค่า (ทำครั้งเดียว)

1

สร้าง Google Sheet ใหม่

คลิกที่นี่ → sheets.new
2

เปิด Apps Script

เมนู Extensions → Apps Script

3

วาง Code นี้ใน editor

// วาง code จากไฟล์ .claude/appscript.gs // หรือกดปุ่มด้านล่างเพื่อ copy
4

Deploy เป็น Web App

→ กด Deploy → New deployment

→ Type: Web app

→ Execute as: Me

→ Who has access: Anyone

→ กด Deploy → Copy URL

5

วาง URL ด้านบน แล้วกด บันทึก
